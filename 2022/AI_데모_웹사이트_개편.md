# AI 데모 웹사이트 개편

**기간**: 2022.01 ~ 2022.09
**역할**: 백엔드 엔지니어 (3명)
**기술 스택**: `FastAPI`, `rabbitmq`, `docker`, `docker-compose`, `MSA`

## 주요 업무

- Message queue 기반 AI 분석 시스템 설계 및 프로젝트 리드
- 신입 백엔드 개발자 교육 및 피드백
- Multi-processing 을 적용하여 GPU 사용 효율 개선
- 동기식 API 지원으로 인터페이스 복잡도를 3배 개선함
- 리소스 제한에 따른 요청 수 관리와 fail 시나리오 설계
