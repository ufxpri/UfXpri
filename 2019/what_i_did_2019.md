# What I Did - 2019

## Year Overview

2019년은 백엔드 엔지니어로서 첫 발을 내딛은 해였다. AI API 개발과 배포를 통해 실제 서비스를 여러 업체에 제공하며 실전 경험을 쌓은 한 해였다.

## Major Project: AI API

**기간**: 2019.01 ~ 2020.01 (13개월)
**역할**: 백엔드 엔지니어 (2명)
**기술 스택**: Flask, TensorFlow, REST API, Async, JWT

### 주요 업무

#### 1. Multi-Client API Deployment
- **REST 방식** API 개발
- **3개 이상의 업체에 동시 배포**
- 각 업체별 맞춤형 배포

#### 2. Authentication System
- **JWT (JSON Web Token) 인증** 구현
- 안정적인 인증 체계 구축
- 보안성 확보

#### 3. GPU Scheduling System
- **배치 단위 처리 시스템** 적용
- 효율적인 GPU 사용 스케줄링
- 리소스 최적화

#### 4. Async API Design
- **비동기 방식 API** 설계
- 분석 단계 추적 가능한 인터페이스
- 진행 상황 모니터링 기능

#### 5. Custom UI Design
- 업체 특징을 고려한 **맞춤형 UI** 설계
- 사용자 중심 인터페이스
- 각 클라이언트 요구사항 반영
