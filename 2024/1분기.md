## 1분기 지표

> 프로젝트 완성률 (%): 월별로 설정된 프로젝트 목표의 달성율을 측정합니다.
> 배포 일정 준수율 (%): 프로젝트 배포 일정의 준수율을 측정합니다.
> 테스트 지표 (회): 시스템의 성능과 안정성을 측정하고 비교하는 테스트 횟수를 기록합니다.
> 리드타임 (일): 작업 시작부터 완료된 결과가 전달되기까지 걸리는 시간을 측정합니다.
> 평균 복구/수리 시간 (MTTR): 오류 발생시점부터 복구까지 걸리는 평균 시간을 기록합니다

### 산출근거
*   **프로젝트 완성률 = (완료된 작업의 수 / 전체 작업의 수) * 100**
    *   수자원 프로젝트: 23/36 = 0.63
    *   알로하 프로젝트: 1/1 = 1.00
    *   미니피씨 프로젝트: 1/2 = 0.50
    *   마이박스 프로젝트: 0/1 = 0.00
    *   총 작업 수: 25/40 = 0.625
*   **배포 일정 준수율 = (기한 내에 완료한 배포 수 / 전체 배포 수) * 100**
    *   소양강 배포 - 3회 시도
    *   성남 정수장 배포 - 지연됨
*   **테스트 지표 = 테스트 테이블 갯수**
    *   [https://docs.google.com/spreadsheets/d/1VxxxHdmo1gRzdkOpyNgBe933dnZftPGfLFKJHYSqc6M/edit?pli=1#gid=248212887](https://docs.google.com/spreadsheets/d/1VxxxHdmo1gRzdkOpyNgBe933dnZftPGfLFKJHYSqc6M/edit?pli=1#gid=248212887)
*   **평균 리드타임 = (모든 작업)(작업 완료 날짜 - 작업 시작 날짜) / 작업의 수**
    *   5, 20, 2, 20, 12, 54, 11, 1, 8
*   **평균 복구/수리 시간 = (총 수리 시간 / 수리 횟수) * 100**
    *   네이버 마이박스 이슈 해결 시간
        *   이슈 시작 시간: 24.03.21 19:22:00
        *   이슈 해결 시간: 24.03.22 14:17:00
